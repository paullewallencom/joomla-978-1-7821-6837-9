Packt Publishing - Learning Joomla 3 Extension Development.
                   Chapter 09 source code
                   Packing Everything Together 
===================================================

This folder contains the source code for the examples in Chapter 9.

* pkg_folio_v1.0.0 - Installation package that includes com_folio v2.7.0 and plg_content_clicktocall v1.2.0
  Use pkg_folio_v1.0.0.zip to install on a Joomla 3 website via extension manager.	

* updates - update server XML file and plg_content_clicktocall_v1.2.1.zip file.
  Place files on update server. Use update manager to upgrade v1.2.0 to v1.2.1.

* com_folio_v2.8.0 - Added additional fields and update sql files.
  Use com_folio_v2.8.0.zip to install on a Joomla 3 website via extension manager.
