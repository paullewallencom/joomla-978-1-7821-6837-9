===================================================
Packt Publishing - Learning Joomla 3 Extension Development.
                   Chapter 03 source code
                   Getting started with Module Development
===================================================

This folder contains the source code for the examples in Chapter 3.

* mod_latestextensions_v1.0.0 - initial version of Latest Extensions module.
  Use mod_latestextensions_v1.0.0.zip to install on a Joomla 3 website via extension manager.

* mod_latestextensions_v1.1.0 - modified version that also supports Joomla 2.5
  Use mod_latestextensions_v1.1.0.zip to install on a Joomla 2.5 website via extension manager.

* mod_latestextensions_v1.2.0 - frontend version of the module.
  Use mod_latestextensions_v1.2.0.zip to install on a Joomla 3 or Joomla 2.5 website via extension manager.

* mod_latestextensions_v1.3.0 - added language files
  Use mod_latestextensions_v1.3.0.zip to install on a Joomla 3 or Joomla 2.5 website via extension manager.

* mod_latestextensions_v1.3.1 - added nl-NL language file
  Use mod_latestextensions_v1.3.1.zip to install on a Joomla 3 or Joomla 2.5 website via extension manager.

* mod_latestextensions_v1.4.0 - added help file
  Use mod_latestextensions_v1.4.0.zip to install on a Joomla 3 or Joomla 2.5 website via extension manager.

* mod_latestextensions_v1.5.0 - added alternative layout support
  Use mod_latestextensions_v1.5.0.zip to install on a Joomla 3 or Joomla 2.5 website via extension manager.

* mod_latestextensions_v1.6.0 - final version that is responsive.
  Use mod_latestextensions_v1.6.0.zip to install on a Joomla 3 or Joomla 2.5 website via extension manager.
